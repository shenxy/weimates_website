## ADDED Requirements

### Requirement: 首页封面 SHALL 体现更新后的企业级价值主张
首页封面 SHALL 使用新的表达方式，将 weiMates 定位为面向企业业务流程的可治理数字员工平台，而不是泛化的 AI 助手或个人提效工具。

#### Scenario: 展示新的封面标题
- **WHEN** a visitor loads the homepage
- **THEN** the hero headline presents the agreed message about 可治理的数字员工进入企业业务流程

#### Scenario: 展示新的封面副标题
- **WHEN** a visitor reads the hero supporting copy
- **THEN** the subtitle explains collaboration, orchestration, governance, process accumulation, and business-value creation in enterprise terms，并以企业级语境解释这些能力

### Requirement: 封面支撑价值点 SHALL 具备明确分工
首页封面 SHALL 包含三个支撑价值点，其标题保持简洁，同时描述内容需要清晰区分运行可靠性、治理能力和长期流程沉淀。

#### Scenario: 三个价值点描述不应语义重叠
- **WHEN** a visitor reads the three supporting value points
- **THEN** each description emphasizes a different concept instead of repeating the same governance idea with different wording

#### Scenario: 三个价值点共同支撑封面主张
- **WHEN** a visitor scans the hero from headline to supporting value points
- **THEN** the supporting value points strengthen the case for using governable digital employees in real enterprise workflows

### Requirement: “平台定位”部分 SHALL 将叙事提升到企业绩效层面
“平台定位”部分 SHALL 明确说明，平台的价值不止于提升个体员工效率，而是致力于提升组织协同效率、业务绩效，以及 AI Native 能力建设。

#### Scenario: “平台定位”部分完成价值层级重构
- **WHEN** a visitor reaches the positioning section
- **THEN** the section explicitly contrasts individual efficiency improvement with enterprise-level performance improvement

#### Scenario: “平台定位”部分引入 AI Native 叙事
- **WHEN** a visitor reads the positioning explanation
- **THEN** the section makes clear that the platform supports the enterprise transition from using AI tools to building an AI Native digital employee system

### Requirement: 实施过程 SHALL 支持分阶段交付
首页封面与“平台定位”部分的刷新 SHALL 按两个阶段交付，以便先验证文案与信息结构，再进行视觉语言和动效系统的重构。

#### Scenario: V1 限定在文案与信息结构调整
- **WHEN** the first implementation pass is executed
- **THEN** it updates the hero and positioning text and any minimal structure needed for the new wording without requiring a full visual redesign

#### Scenario: V2 预留给视觉重构
- **WHEN** the second implementation pass is executed
- **THEN** it may redesign the hero and positioning visual presentation, including layout, graphic language, and motion hierarchy, while preserving the settled narrative
